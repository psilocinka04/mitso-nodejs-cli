# MITSO Node.js CLI Tool

Этот CLI инструмент предоставляет функциональность для двух задач:
1. Кодирование строки - преобразует символы в '(' или ')' в зависимости от их частоты
2. Генерация таблицы умножения - создает таблицу умножения размером NxN

## Установка

```bash
npm install
```

## Использование

Инструмент принимает следующие опции:

- `-i, --input <file>` - путь к входному файлу (необязательно)
- `-o, --output <file>` - путь к выходному файлу (необязательно)
- `-t, --task <task>` - задача для выполнения (обязательно)
  - Доступные задачи: `encode` или `multiply`

### Примеры использования

1. Кодирование строки из файла:
```bash
node src/index.js -t encode -i test/encode_input.txt -o test/encode_output.txt
```

2. Кодирование строки из консоли:
```bash
node src/index.js -t encode
```

3. Генерация таблицы умножения:
```bash
node src/index.js -t multiply -i test/multiply_input.txt -o test/multiply_output.txt
```

### Описание задач

#### Кодирование строки (encode)
Преобразует строку, где каждый символ заменяется на:
- '(' если символ встречается только один раз
- ')' если символ встречается более одного раза

Примеры:
```
Вход: "din" -> Выход: "((("
Вход: "recede" -> Выход: "()()()"
Вход: "Success" -> Выход: ")())())"
Вход: "(( @" -> Выход: "))(("
```

#### Таблица умножения (multiply)
Создает таблицу умножения размером NxN, где N задается во входных данных.

Пример:
```
Вход: 3
Выход: [[1,2,3],[2,4,6],[3,6,9]]
```

### Тестовые данные

В директории `test/` находятся тестовые файлы:
- `encode_input.txt` - примеры строк для кодирования
- `multiply_input.txt` - числа для генерации таблиц умножения

### Примечания
- Если входной файл не указан, программа читает данные из stdin
- Если выходной файл не указан, программа выводит результат в stdout
- Программа продолжает принимать ввод из stdin до прерывания (Ctrl+C)
- Сообщения об ошибках выводятся в stderr
